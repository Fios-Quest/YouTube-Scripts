# Filming Checklist

Preflight

- [ ] Camnera mic is charged (whatever I end up using)
- [ ] Room Lights set to "webcam"
- [ ] Studio Lights are on
- [ ] Unicorn is on
- [ ] Picture frame is on
- [ ] Pipboy is on and showing something interesting
- [ ] Pencil Mic is on
- [ ] Dynamic Mic is off
- [ ] Camera mic is picking up sound

Starting Recording

- [ ] Audacity
- [ ] OBS
- [ ] Camera

Ending Recording

- [ ] Camera
- [ ] OBS
- [ ] Audacity
